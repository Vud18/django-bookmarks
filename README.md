# django-bookmarks
```
Разработана система аутентификации, также добавлена всех необходимые представления для регистрации,
входа в систему и выхода, редактирования и сброс пароля, разработана модель для конкретно-прикладных профилей
пользователя, и создан конкретно-прикладной  бэкенд аутентификации позволяющий пользователям входить в систему,
используя свой адрес электронной почты. Добавлена социальная аутентификация чтобы пользователи могли использовать
свои существующие учётные записи Facebook/Google/twitter/VK для входа в учётную запись на сайте.
```
