# django-bookmarks
```
Разработана система аутентификации, также добавлена всех необходимые представления для регистрации,
входа в систему и выхода, редактирования и сброс пароля, разработана модель для конкретно-прикладных
профилейпользователя, и создан конкретно-прикладной  бэкенд аутентификации позволяющий пользователям
входитьв систему, используя свой адрес электронной почты. Добавлена социальная аутентификация чтобы
пользователи моглииспользоватьсвои существующие учётные записи Facebook/Google/twitter/VK
для входа в учётную запись на сайте.
```
